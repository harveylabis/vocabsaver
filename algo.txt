Pseudo-code for Vocabolary Saver, a dictionary where I can save my words

1. Ask for the word I want to define: word
2. Get the first letter of the word: first_letter
    2.1 Create a function that process the: 
        * first_letter
        * filename based on the first letter: get_filename()
3. Open the file that corresponds to first_letter
    3.1 Create a functions that opens the file
4. Scan through the lines until the word is found
5. Process that line
6. Get the word
7. Get the definition
